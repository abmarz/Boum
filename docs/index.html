Boum is a minimalist programming language designed to help students understand how compilers work. It features its own syntax, parser, semantic analyzer, and interpreter, all built using JavaScript and Ohm-JS.

The goal is to keep the language readable and the architecture understandable, making it ideal for exploring concepts like grammar, scope, type checking, and execution flow.
